# password-manager 存在覆盖密码bug 请勿使用
password-manager 简单密码管理器 使用golang + wails + vue3 开发
采用本地缓存存储数据, 简单
# 前端介绍
 - vue3
 - localforage(本地存储) 
 - spectre.css (UI)

## 环境准备
 - Wails CLI v2.3.1
 - wails v2.0
 - golang 1.18

## 常用指令
```shell

# 开发环境
wails dev

# 打包
wails build
```
